Prefer official Expo skills and docs when Expo is selected.

Check:
- safe areas,
- keyboard behavior,
- navigation/deep links,
- touch targets,
- accessibility,
- reduced motion,
- network/offline states,
- image/media behavior,
- iOS/Android differences,
- build/update strategy.

Do not assume a web component or interaction automatically belongs in native UI.
