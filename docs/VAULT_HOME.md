## Obsidian role

Obsidian is the human cockpit over version-controlled Markdown.

The repository remains the source of truth. Do not create a second private copy of product truth that agents cannot access.

## Recommended links

Use stable requirement IDs and links between:
- PRD requirements,
- acceptance criteria,
- architecture decisions,
- tasks,
- progress evidence.

Example:

`FR-012 → AC-031 → T-044 → evidence → commit`
